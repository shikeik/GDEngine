# GDEngine 开发者手册

欢迎使用 **GDEngine**！这是一个基于 LibGDX 构建的高性能 2D 游戏引擎，专为动作游戏与复杂 UI 交互设计。

## 快速开始

如果你是第一次使用引擎，请先阅读以下章节：

- [引擎架构概览](core/engine_context.md)
- [如何创建你的第一个 Entity](core/ecs_basics.md)

## 核心模块

- **ECS 系统**: 实体、组件与系统的管理。
- **NeonBatch**: 下一代渲染器，支持泛光与高性能批处理。
- **FSM**: 强大的有限状态机，用于 AI 与角色控制。
- **Scripting**: 基于 Java/Lua 的脚本系统。
